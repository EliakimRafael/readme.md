## 🚀 Projetos em Destaque

### 🧼 Limpeza Prime

Desenvolvi um site profissional para a **Limpeza Prime**, uma empresa de serviços de limpeza. O projeto apresenta os principais serviços oferecidos, como limpezas domésticas, escritórios, alojamentos locais e limpezas pós-obras, com um visual moderno, organizado e focado em transmitir confiança aos clientes.

**💻 Tecnologias utilizadas:** HTML, CSS, JavaScript
**📌 Tipo de projeto:** Site institucional / empresarial

### 🎨 EightDesign Web

Criei o site profissional **EightDesign Web**, voltado para a criação de sites modernos e profissionais para empresas, empreendedores e prestadores de serviços. O objetivo do projeto é apresentar soluções digitais, destacar serviços de desenvolvimento web e atrair clientes interessados em ter presença online.

**💻 Tecnologias utilizadas:** HTML, CSS, JavaScript
**📌 Tipo de projeto:** Site profissional / agência web

### 📚 Biblioteca Home

Desenvolvi um banco de dados em **MySQL** chamado **biblioteca_home**, criado para organizar informações relacionadas a livros, autores, categorias e registros de uma biblioteca. O projeto teve como objetivo praticar modelagem de dados, criação de tabelas e consultas SQL.

**🗄️ Tecnologia utilizada:** MySQL
**📌 Tipo de projeto:** Banco de dados

---

### 🎵 musicas_aula_db

Criei o banco de dados **musicas_aula_db** em **MySQL**, voltado para armazenar e organizar informações sobre músicas, artistas, estilos musicais e conteúdos relacionados a aulas. O projeto ajudou no desenvolvimento de habilidades em estruturação de dados e consultas SQL.

**🗄️ Tecnologia utilizada:** MySQL
**📌 Tipo de projeto:** Banco de dados

### 🛒 loja_online

Desenvolvi o banco de dados **loja_online** em **MySQL**, simulando a estrutura de uma loja virtual. O projeto inclui organização de dados como produtos, clientes, pedidos e vendas, com foco em praticar lógica de banco de dados para sistemas comerciais.

**🗄️ Tecnologia utilizada:** MySQL
